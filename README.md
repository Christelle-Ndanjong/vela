# 🌍 VELA — African Talent. Verified. Placed. Global.

VELA is a remote job platform built specifically for African professionals — connecting verified talent to global remote opportunities, with the skills roadmap to get there.

## Live Pages

| Page | Description |
|------|-------------|
| `index.html` | Landing page + waitlist |
| `skills.html` | 15 in-demand skills with roadmaps |
| `jobs.html` | Live remote jobs board |
| `jobs.json` | Auto-updated job data (generated by scraper) |

## How the Job Scraper Works

The scraper pulls remote jobs from three free APIs:
- **RemoteOK** — remoteok.com/api
- **Jobicy** — jobicy.com/api/v2/remote-jobs
- **Remotive** — remotive.com/api/remote-jobs

Jobs are automatically filtered and matched to VELA's 15 skill categories, deduplicated, and saved to `jobs.json`.

## Run Locally

```bash
# Clone the repo
git clone https://github.com/yourusername/vela.git
cd vela

# Run the scraper (no dependencies needed — uses Python stdlib only)
python scraper.py

# Open jobs.html in your browser to see results
```

## Auto-Update with GitHub Actions

The scraper runs automatically every day at 6:00 AM UTC via GitHub Actions. No server needed. Completely free.

To trigger manually:
1. Go to your repo on GitHub
2. Click **Actions** tab
3. Select **Daily Job Scraper**
4. Click **Run workflow**

## Deploy to GitHub Pages

1. Go to repo **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / **root**
4. Save

Your site will be live at `https://yourusername.github.io/vela`
